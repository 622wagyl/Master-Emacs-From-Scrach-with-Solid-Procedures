Meta键(键盘上的option/alter键)
Meta的词源含义是higher, beyond,可以理解为没有最高, 只有更高.
中文译为"元", 发端出, 源头处; So Meta is source, Meta relates to source code.
既然Meta关联src-code, function则是code的first-class-citizen; Meta可以直接调用function.
